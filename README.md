# BitPlace

BitPlace is a simple on-chain version of r/place, mainly built as a first practice project.

Each address can paint up to five pixels for free (aside from gas) per day. Beyond that, you may pay a small fee to paint as many additional pixels as you want. These fees go into a pot, which you have a chance of winning 75% of whenever you paint your 6th pixel each day.

Have fun!

Contact @tenderminttimmy on Twitter for help.
